# Examples

## Basic Form Example

Location: `examples/basic-form`

Demonstrates:

- Schema-driven rendering
- Form submission handling
- Live tool-definition preview

Run:

```bash
npm install
npm run dev
```

## Customer Support Triage Example

Location: `examples/customer-support-triage`

Demonstrates:

- Multi-tool workflow (`classify_ticket`, `suggest_response`, `escalate_ticket`)
- Tool switching and per-tool schema rendering
- Submission history and live definition output

Run:

```bash
npm run dev:triage
```
